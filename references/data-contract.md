# College Hardware Research Data Contract

The scripts use JSON files in `work/`. The agent supplies official public information; the script supplements questionnaire evidence from CollegesChat.

## `schools.json`

Generated by:

```powershell
conda run -n base python scripts/run.py extract-schools --input input\志愿表.xlsx
```

It is an ordered array of school names.

## `official_sources.json`

The agent writes an array:

```json
[
  {
    "school": "青岛大学",
    "level": "公办综合类本科；山东省属重点高校",
    "location": "山东省青岛市",
    "campuses": "浮山校区、金家岭校区、松山校区",
    "campus_addresses": "浮山校区：山东省青岛市崂山区宁夏路308号；金家岭校区：山东省青岛市崂山区松岭路93号；松山校区：山东省青岛市市北区登州路38号",
    "source": "https://www.qdu.edu.cn/xxgk/xxjj.htm"
  }
]
```

Rules:

- Leave unknown fields blank.
- Use `；` between multiple campuses.
- Do not put CollegesChat or unofficial Q&A pages in `source`.
- Do not write speculative words such as `可能`, `据说`, or `大概` into official fields.

## CollegesChat Field Mapping

- `生活条件`: take evidence from traffic, delivery, supermarket, dining, and free-text questions.
- `有无空调`: `教室和宿舍有没有空调？`
- `宿舍几人间`: `宿舍是上床下桌吗？`
- `有无独立卫浴`: `有独立卫浴吗？没有独立浴室的话，澡堂离宿舍多远？`
- `有无早晚自习`: `有早自习、晚自习吗？`
- `有无早操`: `有晨跑吗？` and `每学期跑步打卡...`
- `宵禁情况`: `现阶段学校的门禁情况如何？` and night-check or dorm-lock questions.
- `到点是否断电断网`: `每天断电断网吗...` and `宿舍限电情况？`

## Workbook Rule

Each summary cell should be readable without opening raw evidence:

```text
六人间-没有独立卫浴-澡堂距离见摘录；近年问卷摘录（68条）：有独立卫浴吗？...（2026-06）；学校允许点外卖吗？...（2026-06）
```

Do not include CollegesChat answer IDs. Keep the question text and month.
